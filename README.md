Allows the old "Orange" screenmate from the Windows 95/98/xp era to run on modern windows 7/8/10/11.<br>
<br>
All the animations and sounds function! You can bring back that classic laugh!<br>
<br>
To exit app simply catch the orange and double click on him, you can turn off the sound. or mute him, and control gravity.<br>
<br>
<br>
Installation:<br>
With MSI file.<br>
Run the .MSI installed.<br>
Install to C:\Program Files\<br>
Must install into "Program Files" not (x86) for the start menu shortcut to function.<br>
<br>
Manual:<br>
Download source code<br>
Make new folder called "Orange" in "Program Files"<br>
Paste source into Orange folder.<br>
<br>
If you install to any other folder you will need to edit the start menu shortcut to point to the new location.<br>
You will also need to edit the "Orange.bet" file to point to the folder you installed to.<br>
